IExecutorG, F InterfaceSingular.Evolution Library
Interface for executors. Executors takes care of the execution of certain tasks between the framework. For example, it can provide a multithreaded environment for long running tasks of the algorithm, in example, the fitness calculations. Also, it can provide caching, pausing, stopping and resuming of those tasks.
