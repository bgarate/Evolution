GaussianBaseMutatorG, F ClassSingular.Evolution Library
A mutator which applies on IListGenotypeG, R. The result of this mutator applied over a A FloatGene is a new B FloatGene (with same boundaries) and a new value with a Gaussian distribution for unbounded genes, or BoundedGaussian for bounded and mean A.Value with one standard deviation for unbounded genes BoundedGaussian for bounded genes)
